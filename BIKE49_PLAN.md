# Bike49 MVP Project Plan

## Project Overview
* Bike49 is a bike buying and selling MVP platform
* Inspired by Cars24 and Spinny
* Built using Next.js + Tailwind CSS
* No backend, only frontend simulation

## Pages
* Home
* Buy
* Sell
* About
* Contact

## Features
* Bike listing with full details
* Filters (price, city, ownership, year)
* Sorting (low-high, high-low, newest)
* Sell form (basic)
* AI features (chatbot, price estimator, recommendation)
* EMI calculator
* Insurance UI
* Challan UI

## UI Style
* Blue gradient hero
* Dark overlay with bike images
* Clean startup-like design
* Card-based layout
